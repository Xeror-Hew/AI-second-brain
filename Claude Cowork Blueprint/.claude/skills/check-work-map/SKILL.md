Before you work in a part of the workspace you don't know:

1. Open `project_brain/work_map/work_map_index.md` for the lay of the land and the fragment table.
2. Find the fragment whose description matches what you need and read it.
3. Then go to the material itself.

No map, or clearly stale? Run `/work-map` first. The map gets you to the right area faster, then you dig in.
